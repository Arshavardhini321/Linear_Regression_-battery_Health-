# Battery Health Prediction using Simple Linear Regression

## Overview
This project predicts the battery health of an electric vehicle based on the total kilometers driven using Simple Linear Regression.

## Dataset
- Feature: Kilometers_Driven
- Target: Battery_Health

## Algorithm
Simple Linear Regression

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
- Data Loading
- Data Exploration
- Data Visualization
- Correlation Analysis
- Train-Test Split
- Model Training
- Prediction
- Model Evaluation

## Results
The model successfully predicts battery health with a strong linear relationship between kilometers driven and battery health.
